# 一些变量作用

## 控制显示隐藏

### showSearch -- 控制查询条件显隐
存储位置：vuex
变量名：showSearch
主体：Content.vue -- 查询条件
变量作用位置：Content.vue
改变状态位置：RouterList.vue SideBar.vue -- 路由切换时默认隐藏
变换显隐位置: Bread.vue -- method: searchBtn

### isBread -- 控制所选查询日期 控制查询条件的显隐的按钮 的显隐
存储位置：Search.vue
变量名：isBread
主体：Bread.vue
变量作用位置：Search.vue
改变位置：Search.vue -- 默认为 true
改变方法：通过检测路由的变化（$route）,当路由的 name 值为 welcome 时，值改变为false
